# Happy Phone

Мини-приложение для оформления заявок на доставку посылок.

## Стек

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Zod (валидация)

## Запуск

```bash
npm install
npm run dev
```

## Функционал

**Форма заявки** (`/`)

- 3 шага: отправитель → получатель → подтверждение
- Валидация полей
- Маска телефона

**История заявок** (`/orders`)

- Список с поиском и фильтрами
- Детали заявки
- Удаление с подтверждением

Данные хранятся в localStorage.
